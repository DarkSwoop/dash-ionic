## Ionic docset generator for dash

Prerequisites: ruby, bundler

Generation:

1. ```git clone https://github.com/DarkSwoop/dash-ionic.git```
2. ```cd dash-ionic```
3. ```bundle install```
4. ```rake```

The docset will be created in the ```dist``` folder.
